### Getting Started
All of the required packages are listed in `requirements.txt`. Running the command `pip install -r requirements.txt` will download all packages necessary to run *Bird Storm*.

### Controls
W: Move up\
A: Move to the left\
S: Move down\
D: Move to the right\
Space Bar: Attack
